what I Think

  • 퍼블리싱 기본기

    인덴트,주석, HTML과 CSS의 속성을 고려하는 등 기본을 놓치지 않고 습관화하여 팀으로 협업할 때 착오 없이 진행할 수 있도록 노력하고 있습니다.

  • 기획을 고려한 퍼블리싱

    직접 기획서를 작성해 보면서 UI/UX 측면도 함께 고려하였습니다. 심미성 뿐이 아닌 편의성도 고려한 퍼블리싱을 중요하게 생각합니다.

  • 체계적인 퍼블리싱

    와이어프레임 구조를 먼저 생각합니다. 한 태그를 쓸때도 시멘틱 태그를 생각하고, CSS와 제이쿼리가 상호작용하는 부분도 함께 고려하며 작업합니다.

  • 빠른 적응력

    SCSS, git 등 실무에 필요한 부분을 배워 적용하려 노력합니다. 부족한 부분은 인정하고 필요한 부분을 배워 실무에 필요한 인재가 되겠습니다.

Skills

  • H
    HTML

    웹 표준과 웹접근성을 준수하는 HTML 구조설계를하기 위해 고심합니다.
    특히 웹표준을 위해 HTML5에서 권장하는 마크업 방식과 시멘틱 태그를 사용합니다.
    클래스 네임과 정보구조설계를 꼼꼼히하고 작업을 진행합니다.
    전체적인 폰트서식, 컬러 가이드, 와이어프레임 등을 미리 짠 후 체계적인 퍼블리싱을 수행합니다.

  • C
    S
    CSS/SCSS(SASS)

    크로스브라우징을 위해 안정적인 CSS속성을 사용해서 디자인합니다. 최적화를 위해 불필요한 플러그인은 최소화하고 CSS로 구현을 우선시합니다.
    네스팅, 컴포넌트화,mixins 등 빠른 업무를 위해 SCSS를 사용하여 퍼블리싱을 합니다.

  • RW
    Responsive Web

    반응형 웹 제작에 있어서 크로스브라우징을 위해 float,position,media Query를 사용할 줄 알며 flex를 활용한 웹페이지 제작도 가능합니다.

  • JS
    JQuery

    네비게이션,탭 메뉴, 어코디언 등 퍼블리싱에 필요한 기능적인 부분을 순수한 제이쿼리 코딩으로 제작합니다.
